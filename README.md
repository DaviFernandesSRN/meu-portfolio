Meu Portfólio - Front-end Developer
Bem-vindo ao repositório do meu portfólio pessoal! Este projeto marca a minha transição de carreira de Analista de Suporte Sênior para Desenvolvedor Front-end. Aqui, apresento minhas habilidades em criar interfaces modernas, responsivas e de alta qualidade.

🚀 Sobre o Projeto
O objetivo deste portfólio é servir como um cartão de visitas digital, detalhando minha experiência técnica e os projetos que venho desenvolvendo, como o Livraria Macabeia e o TalentMatch.

🛠️ Tecnologias Utilizadas
Este projeto foi construído utilizando as seguintes tecnologias:

HTML5: Estruturação semântica do conteúdo.

CSS3: Estilização avançada com Flexbox para layouts dinâmicos.

Google Fonts: Utilização das fontes Krona One e Montserrat para uma tipografia moderna.

🎨 Design
O layout foi planejado para passar uma imagem profissional e tecnológica.
